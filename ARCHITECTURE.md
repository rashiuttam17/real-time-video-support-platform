# Forge 2 Multi-Agent Architecture

## Agent Layout
* **Orchestrator (Brain):** Hermes Agent tracking context and persistent memory.
* **Execution (Hands):** OpenClaw executing backend/frontend files.

## Slack Communication Scheme
* #sprint-main: Strategic planning and human validation loop.
* #agent-coder: Technical task routing and direct code execution.
* #agent-log: Telemetry logs and automated cron reports.
